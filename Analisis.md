Eelemento React.- Componente
¿Donde lo usas? paar dividir interfaz en partes que sean independientes como app.jsx
¿Por que es el adecuado? por que es facil de entender y utilizar

--------------------------

Eelemento React.- JSX
¿Donde lo usas? para escribir una estructura similar a html
¿Por que es el adecuado? por que puede definir cosas con javascript

--------------------------

Eelemento React.- Props
¿Donde lo usas? para enviar componente 1 a componente 2
¿Por que es el adecuado? por que separa los datos entre componentes

--------------------------

Eelemento React.- Estado (usestate)
¿Donde lo usas? para guardar valores que dambian con lo que haga el usuario
¿Por que es el adecuado? por que actualiza autimaticamente la vista cuando cambia 

--------------------------

Eelemento React.- Renderizado de listas (.map + key)
¿Donde lo usas? para generar cosas dinamicas usando un array de base
¿Por que es el adecuado? por que usa el componente key y ayuda a react a identificarlos

--------------------------

Eelemento React.- Renderizado condicional
¿Donde lo usas? para mostrar cpsas distintas segun una condicion
¿Por que es el adecuado? permite cambiar o modificar la pagina sin recargarla o reiniciar el servidor
